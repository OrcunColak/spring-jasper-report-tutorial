# DataSources

JDBC : JRResultSetDataSource
JavaBean : JRBeanCollectionDataSource, JRBeanArrayDataSource
Map-based : JRMapCollectionDataSource, JRMapArrayDataSource
Table Model : JRTableModelDataSource
XML : JRXmlDataSource
CSV : JRCsvDataSource
XLS : JRXlsDataSource
Empty : JREmptyDataSource
